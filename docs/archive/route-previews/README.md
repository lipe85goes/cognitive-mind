# Route Preview Archive

PNG previews generated for Rota Estratégica GLB development were moved here in CLEAN-05.

They are not loaded by the app at runtime. The active runtime assets remain in `public/models/route/*.glb` plus the board texture sources in `public/models/route/textures/`.

If a visual preview is needed again, run the relevant Blender generator with `--preview`; by default those scripts recreate previews under `public/models/route/`.
