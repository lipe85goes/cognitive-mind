# WORLD-COHESION-01 review material

This folder contains review-only evidence for the Home-to-world visual bridge.

- `world-cohesion-review.png` compares the two primary worlds promoted into the shared visual contract.
- Runtime variants live in `public/illustrations/home/`.
- Source renders remain in `public/illustrations/worlds/` because they are still part of the existing visual asset library.

Recreate the optimized runtime variants and this review sheet with:

```powershell
node tools/assets/create_world_cohesion_assets.mjs
```

Do not reference this folder from runtime code.
