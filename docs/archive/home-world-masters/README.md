# Home World Masters

This folder stores non-runtime review material for HOME-ART-02.

- Runtime assets live in `public/illustrations/home/*.webp`.
- `home-world-review.png` is a review sheet only and must not be referenced by the app.
- Recreate the current runtime sprite set with `node tools/assets/create_home_world_set.mjs`.
- Keep source/master/review files outside `public/` unless they are intentionally used at runtime.
