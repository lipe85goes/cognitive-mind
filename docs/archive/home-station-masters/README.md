# Home Station PNG Masters

This folder archives the original heavy PNG station images used before ASSET-01.

These files are not runtime assets. The runtime versions live in:

- `public/illustrations/station-memory.webp`
- `public/illustrations/station-route.webp`
- `public/illustrations/station-command.webp`
- `public/illustrations/station-logic.webp`
- `public/illustrations/station-garden.webp`

The PNG masters were kept here so the artwork can be recompressed or inspected
later without keeping multi-megabyte files in `public/`.

Do not move these files back to `public/` unless there is a deliberate asset
pipeline decision to ship PNG masters again.
